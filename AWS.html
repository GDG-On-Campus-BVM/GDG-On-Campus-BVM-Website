About AWS Academy 2023!

AWS Academy program provides on opportunity to kickstart your carrier in cloud and get hands-on practice on Amazon Web services - the tool that powers top website like amazon, Netflix, Linkedin, Twitch etc.

Along the way, you will learn & practice concepts like computing, application development, big data & machine learning using cloud.
